# Safety and Governance

This prototype is safety-first but not clinically validated.

## Guardrails included

- PII redaction before logging/downstream AI use
- prompt-injection detection
- protocol evidence retrieval
- graph-based danger-sign routing
- emergency supervisor review queue
- citation verifier warning
- missing information prompts

## Required before real deployment

- Ministry-approved protocol sources
- clinical validation by qualified reviewers
- human-in-the-loop review policy
- data protection impact assessment
- audit log review
- secure hosting and access controls
- field usability testing with HEWs
- incident response process
